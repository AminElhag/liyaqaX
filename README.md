# Liyaqa

Monorepo for the Liyaqa platform.

## Repository layout

| Directory | Description |
|---|---|
| `backend/` | Spring Boot / Kotlin API |
| `web-nexus/` | Internal platform dashboard |
| `web-pulse/` | Club operations dashboard |
| `web-coach/` | Trainer dashboard |
| `web-arena/` | Member self-service portal |
| `mobile-arena/` | Member mobile app (Android + iOS) — git submodule |

## Local development

See the root `CLAUDE.md` for full setup instructions, service ports, and common commands.
